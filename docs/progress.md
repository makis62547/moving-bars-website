# Fortschritts-Checkliste

Verwende separate Branches pro Abschnitt, um inkrementell an der Seite zu arbeiten. Vorschläge in Klammern.

- [ ] Navigation & Header (`feature/nav-header`)
- [ ] Hero-Bereich (`feature/hero-layout`)
- [ ] Erster Content-Block / Intro (`feature/block1-intro`)
- [ ] Leistungen / Module (z. B. Cocktail-Catering, Mobile Bar) (`feature/block-services`)
- [ ] Bildgalerien & Testimonials (`feature/block-gallery-testimonials`)
- [ ] Kontakt / Footer (`feature/contact-footer`)
- [ ] Feinabstimmung Typografie & Spacing (`feature/typo-spacing`)
- [ ] Finale visuelle Regression & Review (`feature/final-vr-check`)
